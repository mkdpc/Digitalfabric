Target Digital Fabric QA Checker v1.5

Local browser-based checker for .zfab naming, required texture maps, PNG resolution, DPI, physical size, and package consistency. v1.5 adds Target naming convention audit: FASID_FabricType_Weightgsm_Content%_FiberAbbreviation, including gsm requirement, content total = 100, and approved fiber abbreviation validation.

Note: This tool first tries ZIP package reading. If the .zfab is not ZIP-readable, it will scan for embedded PNG textures and still check resolution, DPI, physical size, and 8-bit PNG where possible.

Naming rule v1.5: FASID_FabricType_Weightgsm_Content%_FiberAbbreviation. Example: FAS1234_Jersey_180gsm_80_10_10_RecCtn_RecPoly_Span. Weight must include gsm; content numbers must total 100; fiber names must use approved abbreviations.
